# **Maven**

Maven is build tool used to build programs, it internally manages the dependencies which saves our time.

pom.xml 

Some dependencies for example

- JDBC Driver: used to connect programs to the databases such as SQL, MySQL
- SL4J: Logger - used to log messages

**Build Tools**

There are two tools used to compile java code

Maven: Is mostly used 

Gradle: Is mostly used in Android

**Why use build tool?**

- It automatically downloads, installs and manages dependencies internally/behind the scenes
- Run custom variables on run
- Automatically creates **Jar OR Var** file
- Deploys automatically
    - Maven deploy: This command is used to deploy the files
    - It also handles the versioning automatically
